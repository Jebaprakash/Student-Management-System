# Student Management System (Node.js + PostgreSQL + EJS)

A web-based student management system built using Node.js, Express, PostgreSQL, and EJS.  
This system allows registration, authentication, student data management, attendance tracking, and marks management with grading logic.

## Features

- User registration and login authentication  
- Add, edit, and delete student records  
- Manage student marks and calculate grades automatically  
- Attendance management interface  
- JWT-based session token generation  
- EJS templating for front-end rendering  

---

## Technologies Used

| Component | Technology |
|------------|-------------|
| Backend | Node.js, Express |
| Database | PostgreSQL |
| Frontend | EJS Templates |
| Language | TypeScript |
| Libraries | pg, body-parser, jsonwebtoken, ejs, path, fs |
